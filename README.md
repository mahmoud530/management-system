# Employee Management System

A simple **PHP & MySQL** based employee management system with authentication (Login/Register), user roles, and employee data management.

---

## üöÄ Features
- User Authentication:
  - Register with profile image upload.
  - Login/Logout system with sessions.
- Roles:
  - `admin` / `super_admin`: full access.
  - `user`: limited access to profile only.
- Employees Management:
  - Add employees (linked with user accounts).
  - Upload and display employee profile images.
  - Search employees by **name** and filter by **salary/department**.
  - Live search with AJAX.
- Dashboard:
  - Show total users, employees, and departments.
  - Different UI based on role.
- Profile Page:
  - View personal info and department.
  - Edit account (optional).
